# Foodie Online Food Delivery Website

A premium, fully functional food delivery website built with HTML, CSS, and JavaScript.

## Features
- Dynamic menu with category filtering.
- Global search for dishes.
- Cart system with real-time updates and total calculation.
- Responsive design for all devices.
- Modern aesthetics with glassmorphism and animations.

## How to Deploy to GitHub Pages
1. **Push your code to a GitHub Repository**:
   - Initialize a git repo: `git init`
   - Add files: `git add .`
   - Commit: `git commit -m "Initial commit"`
   - Create a repo on GitHub and follow the instructions to push your code.

2. **Enable GitHub Pages**:
   - Go to your repository on GitHub.
   - Click on **Settings**.
   - On the left sidebar, click on **Pages**.
   - Under **Build and deployment**, set the source to **Deploy from a branch**.
   - Select the `main` branch and the `/ (root)` folder.
   - Click **Save**.

3. **Wait for Deployment**:
   - Your site will be live at `https://your-username.github.io/your-repo-name/` within a few minutes.

## Important Note for Deployment
- Ensure that the file structure remains as follows:
  ```
  / (root)
  ├── index.html
  ├── css/
  │   └── style.css
  └── js/
      └── app.js
  ```
- All links in `index.html` are relative, ensuring that the site works perfectly on GitHub Pages.
